📚 Documentation

Complete guide to WhatsApp Contact Exporter

🚀 Quick Start

Get started with WhatsApp Contact Exporter in just a few minutes.

Installation

  1. Download the extension from the homepage
  2. Extract the ZIP file to a folder
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode"
  5. Click "Load unpacked" and select the folder

First Use

  1. Go to web.whatsapp.com
  2. Click the extension icon in your toolbar
  3. Click "Extract Contacts"
  4. Choose your export format (CSV, JSON, or Clipboard)

📖 Features

Contact Extraction

The extension extracts the following information for each contact:

  • Name - Contact or group name
  • Phone Number - When available (may show "N/A" for saved contacts)
  • Group Status - Whether it's a group chat or individual
  • Last Message - Preview of the most recent message
  • Timestamp - When the last message was sent

Export Formats

CSV (Comma-Separated Values)

Perfect for spreadsheet applications like Excel or Google Sheets.

Name,Phone Number,Is Group,Last Message,Timestamp
John Doe,+1234567890,No,Hey there!,10:30 AM
Family Group,N/A,Yes,See you tomorrow,Yesterday

JSON (JavaScript Object Notation)

Structured data format ideal for developers and database imports.

[
  {
    "name": "John Doe",
    "phoneNumber": "+1234567890",
    "isGroup": false,
    "lastMessage": "Hey there!",
    "timestamp": "10:30 AM",
    "index": 1
  }
]

Clipboard

Tab-separated format for quick pasting into documents.

⚙️ Options

Include Group Chats

Toggle whether to include group conversations in the export. Enabled by default.

Scroll to Load All Chats

Automatically scrolls through your chat list to ensure all contacts are loaded before extraction. Recommended for large contact lists (100+ chats).

🔒 Privacy & Security

100% Private: All processing happens locally in your browser. No data is ever sent to external servers.

What We Access

  • WhatsApp Web chat list (only when you click "Extract")
  • Contact names and phone numbers visible on the page
  • Last message previews (already visible in chat list)

What We DON'T Access

  • Message content (beyond what's visible in previews)
  • Media files (photos, videos, documents)
  • Your phone number or account details
  • Any data from other websites

Permissions

  • activeTab - Read WhatsApp Web page when you use the extension
  • storage - Save your preferences (less than 1 KB)
  • web.whatsapp.com - Only works on WhatsApp Web

🐛 Troubleshooting

Extension Not Appearing

Solution: Make sure Developer mode is enabled in chrome://extensions/. Try reloading the extension or restarting Chrome.

No Contacts Found

Solution: Enable "Scroll to load all chats first" option. Make sure you're logged into WhatsApp Web and have active conversations.

Phone Numbers Show "N/A"

Note: This is normal for saved contacts. WhatsApp Web shows contact names instead of phone numbers. Unsaved contacts will display their phone numbers.

Extraction Seems Incomplete

Solution: Enable auto-scroll option and wait for it to complete. For very large contact lists (500+), you may need to manually scroll first.

💡 Tips & Best Practices

  • Before extracting: Scroll through your chat list to load all contacts, or enable the auto-scroll option
  • For large lists: Enable "Scroll to load all chats first" and be patient (may take 30-60 seconds)
  • Regular backups: Run the extraction periodically to keep your contact backup up-to-date
  • CSV for spreadsheets: Use CSV format if you plan to open in Excel or Google Sheets
  • JSON for developers: Use JSON format for database imports or custom processing

🔧 Technical Details

Browser Compatibility

  • Google Chrome (latest version)
  • Microsoft Edge (Chromium-based)
  • Brave Browser
  • Any Chromium-based browser

Performance

  • Extraction Speed: ~100-200 contacts per second
  • Memory Usage: Less than 50 MB
  • Extension Size: ~30 KB

Limitations

  • Only works on WhatsApp Web (not mobile app)
  • Phone numbers may not be available for all contacts
  • Group member details are not extracted (only group names)
  • Requires active WhatsApp Web session
